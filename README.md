# Online Ticket Booking Platform - Client

## Purpose
Frontend for the Online Ticket Booking Platform. Built with React, Vite, Tailwind CSS, and BetterAuth.

## Live URL
https://online-ticket-booking-platform-clie-sigma.vercel.app/

## Key Features
- BetterAuth authentication (email/password + Google OAuth)
- Three user roles: User, Vendor, Admin
- Dark/Light mode toggle
- Search, filter, sort, and pagination for tickets
- Countdown timer for departure dates
- Stripe payment integration
- Vendor revenue charts (Recharts)
- Responsive design for all devices
- Loading spinners and error pages

## NPM Packages Used
- react, react-dom - UI framework
- react-router-dom - Client-side routing
- better-auth - Authentication client
- axios - HTTP client
- tailwindcss - Utility-first CSS
- swiper - Hero slider
- recharts - Charts for revenue overview
- react-hook-form - Form management
- @stripe/react-stripe-js, @stripe/stripe-js - Payment UI
